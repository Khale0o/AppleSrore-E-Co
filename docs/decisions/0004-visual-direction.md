# ADR 0004: Visual Direction

## Context

The first vertical slice needs a recognizably premium experience without imitating Apple or relying on generic commerce patterns.

## Decision

Use a dark, product-led cinematic direction with graphite, silver, soft white, and restrained cool-blue lighting. Build one connected smartphone journey with purposeful transitions, minimal chrome, and reduced-motion alternatives.

## Alternatives considered

- Bright, grid-first marketplace layouts.
- Gold-led luxury styling.
- Card-heavy glassmorphism and decorative animation.

## Consequences

Composition and assets carry more of the experience, so image quality, animation restraint, and performance budgets are critical. The identity must remain original and unaffiliated with Apple Inc.

## Revisit conditions

Revisit when accessibility testing, performance profiling, or approved product categories demonstrate a need to adjust the visual system.
