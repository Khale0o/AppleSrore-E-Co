# Product Scope

## Approved direction

AppleStore is a premium cinematic technology-commerce experience for an ecosystem of smartphones, laptops, tablets, smart watches, earbuds, and accessories. It is not a traditional grid-first store.

The visual direction is minimal, cinematic, and product-focused: large typography, generous spacing, primarily dark scenes, and silver, graphite, white, and subtle cool highlights. Gold must not be the primary accent. The application should feel like an interactive product film, not a visible collection of generic UI cards.

## First approved vertical slice

```text
Splash
→ Cinematic Home Hero
→ Direction-aware Product Carousel
→ Product Details
→ Color and Storage Configuration
→ Add to Bag
```

Use one primary smartphone, a secondary smartphone for carousel testing, and supporting ecosystem imagery only where needed.
